# ML Projects

## Projects

- Predicted California house prices using Linear Regression
- Dataset: California Housing (20,640 rows)
- R² Score: 0.61
- Tech: Python, Pandas, Scikit-learn, Matplotlib

### 2. Titanic EDA
- Exploratory Data Analysis on Titanic survival dataset
- Analyzed survival rates by gender, class, and age
- Tech: Python, Pandas, Matplotlib, Seaborn

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
